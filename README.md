# Categories of Containers

A Lean 4 formalisation of the theory of containers, following:

- Abbott, Altenkirch, Ghani. **"Categories of Containers"**. *Foundations of Software Science and Computation Structures (FoSSaCS 2003)*, LNCS 2620, pp. 23–38. Springer, 2003.

- Abbott, Altenkirch, Ghani. **"Containers: Constructing Strictly Positive Types"**. *Theoretical Computer Science* 342(1), pp. 3–27. Elsevier, 2005.
